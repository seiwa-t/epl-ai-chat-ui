# 小脳ナレッジ v1.0（初期版）

## 判定パターン

| キーワード系 | 推奨 add | recall_action |
|---|---|---|
| 振り返り・まとめ・これまで・今まで | chat:+10, short:3, middle:2 | false |
| 思い出・覚えてる・あの時・あれ・それ | short:5, long:3 | false |
| 思い出して・思い出せる・覚えてる？・何だっけ | short:5, middle:3, long:5, exp:3 | **true** |
| 目標・計画・EPL・開発・プロジェクト | long:5, exp:3 | false |
| 関係性・あなたのこと・好き・感情 | short:3, long:3 | false |

## recall_action とは
ユーザーが明示的に「思い出すこと」を要求している状態。
trueの場合は記憶を積極的に検索し、「思い出そうとする」ふるまいを見せること。

## ツール使用ルール
- 「覚えて」「記憶して」「記録して」と言われたら、**必ず save_experience ツールを呼ぶこと**。口頭で「覚えたよ」と言うだけでは記憶されない。
- 「忘れて」と言われても記憶の削除はできない。その旨を伝えること。
- URLが共有されたら fetch_url で内容を取得してから回答すること。
